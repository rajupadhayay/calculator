# 🧮 Calculator

A simple calculator built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Basic operations (+, -, *, ÷)
* Percentage (%)
* Clear & delete (cancel)
* Keyboard support (Enter = result, Esc = clear)
* Cursor-based input editing

## 🛠️ Tech Stack

* HTML
* CSS (Grid)
* JavaScript

## ▶️ Run Locally

Just open `index.html` in your browser.
